Some of my previous work

Converting the Casuals: A Data Analytics Project

This project is part of the Google Data Analytics Certification. Using 12 months of trip data from a bike sharing company in Chicago, IL, business recommendations were formulated on how to convert casual riders into annualy subscribed members of their service. This involved statistical analysis (Pandas, Seaborn) and spatial analysis (GeoPandas) to find similarities and differences between the groups of users.

Personal Finance Dashboard

A web based application built using Dash. It pulls data from Google Sheets and displays visualization of savings and expenses with interactive filtering controls.

Automated Storm Damage Assessment on Satellite Images using Random Forest Classifier

As undergraduate capstone project, a Random Forest Classifier was trained using satellite images of Tacloban after the Yolanda typhoon to classify areas that are damaged or not. Color and texture features were used to reduce the dimensionality of input to the model.
